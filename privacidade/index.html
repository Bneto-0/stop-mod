Politica de privacidade

Os dados digitados na loja sao usados para autenticar o cliente, registrar pedidos e concluir pagamentos. Informacoes sensiveis ficam sob responsabilidade do backend e do gateway configurado.

  • Dados de cadastro ficam vinculados a conta do cliente.
  • Pagamentos sao iniciados via PagBank conforme a configuracao atual.
  • Endereco, pedidos e notificacoes podem ser armazenados localmente no navegador para melhorar a experiencia.
